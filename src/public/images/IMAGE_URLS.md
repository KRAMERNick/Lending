# Изображения для сайта

Загрузите все изображения в эту папку `/public/images/` со следующими именами:

## Список необходимых файлов:

### Главная страница (Hero):
- **hero-photo.png** - Главное фото Николая Бокарева

### О преподавателе (About):
- **profile-photo.png** - Портретное фото
- **video-cover-1.png** до **video-cover-6.png** - Обложки видео с уроков

### Галерея учеников (Gallery):
- **student-photo-1.png** до **student-photo-3.png** - Фото учеников
- **student-video-cover-1.png** до **student-video-cover-6.png** - Обложки видео учеников

### Студия (Studio):
- **studio-photo-1.png** до **studio-photo-3.png** - Фото студии Chayka Studio

### Игра (DrumGame):
- **drum-set.png** - Изображение барабанной установки

### Контакты (Contact):
- **telegram-icon.png** - Иконка Telegram
- **whatsapp-icon.png** - Иконка WhatsApp

### Дополнительно:
- **/public/favicon.ico** - Иконка сайта для вкладки браузера

---

## Как использовать изображения с GitHub:

Если ваши изображения лежат на GitHub, используйте прямые ссылки:

```
https://raw.githubusercontent.com/ВАШ_ЮЗЕРНЕЙМ/ВАШ_РЕПОЗИТОРИЙ/main/images/hero-photo.png
```

Или загрузите файлы напрямую в эту папку.
